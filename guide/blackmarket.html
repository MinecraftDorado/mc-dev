Commands

  • /bm sell [value]

    - Sell a item

  • /bm reload

    - Reload config

  • /bm setnpc (skin_name)

    - Set a NPC

  • /bm removenpc

    - Remove a NPC

  • /bm open

    - Open the BlackMarket

Permissions

  • blackmarket.admin

    - Allow use all commands

    • blackmarket.open

      - Allow use open command

    • blackmarket.reload

      - Allow use reload command

    • blackmarket.npc

      - Allow use setnpc and removenpc commands

      • blackmarket.setnpc

        - Allow use setnpc command

      • blackmarket.removenpc

        - Allow use removenpc command

    • blackmarket.limit

      - Unlimited items in sell

    • blackmarket.limit[number]

      - Limited items in sell
